# UNN Markeptplace (UNNify)

A custom e-commerce solution built exclusively for the University of Nigeria, Nsukka (UNN) community. This platform allows members of the Lion community to:

- Discover great deals  
- Sell items effortlessly  
- Connect with trusted buyers and sellers  



## Features
  
- User verification using **school email**  
- Open to **all UNN community members**, not just students  
- Simple and intuitive interface  
- Secure PHP & MySQL backend  
- Campus-focused marketplace for trust and safety  

---

## Tech Stack

- **Backend:** PHP  
- **Database:** MySQL  
- **Authentication:** School email verification  

---

## Getting Started

1. Clone the repository:


git clone https://github.com/Major-Kosdan/campus-buy-and-sell-platform.git

2. Place the project in your htdocs folder (if using XAMPP).


3. Set up your MySQL database and update the configuration file.

4. Open your browser and navigate to http://localhost/project-folder.
